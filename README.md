## Goals

* `bx proba test/there/`
* `bx proba test/there/that.rb`
* `bx proba test/there/that.rb:123`
* `bx proba test/there/that.rb:123:210`
* `bx proba test/there/that.rb:123-210`
* dots
* colors
* times monow
